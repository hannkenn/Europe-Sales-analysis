# Europe Sales Data Analysis
## Overview
This project analyzes European sales data. Raw data is cleaned, transformed in python/Jupyter, and loaded into SQL Server for downstream analysis.
## Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook
SQL Server
## Workflow
Load raw CSV data from 'data/raw'
clean and transform: fix dates, handle missing values, create Year/Month features
Load cleaned data into SQL Server
Perform analysis and visualization
## key Insights
Top revenue-generating countries
yearly revenue trends
key product catagories
